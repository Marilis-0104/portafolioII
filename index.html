TIPOS DE ETIQUETAS

En HTML y CSS tenemos diferente tipos de etiquetas cada uno con un propósito en particular

Una etiqueta se puede componerse de dos maneras:

ESTRUCTURA DE ETIQUETAS
Etiquetas que solo aperturan Etiquetas que tinen apertura y cierre
Este tipo de etiqueta común mente se utilizan para indicar una dirección u origen. Se componen por argumento u opciones tales como src href. Este tipo de etiqueta comunmente se utilizan para indicar una dirección u origen. Se componen por argumento u opciones tales como src href.

LISTA DE ETIQUETAS DIRIGIDAS AL TEXTO

TITULOS

Hay hasta 6 etiquetas en HTML para definir títulos o secciones. Todas ellas llevan por defecto el texto en negrita, y son:
<h1> <h2> <h3> <h4> <h5> <6>.

Es posible crear titulos para head y para body


HEAD

  • <title></title>Permite darle un título a una página web. Este título se puede encontrar en la barra de título del navegador, así como en las páginas de resultados de los buscadores

BODY

  • <h1></h1> etiqueta que se utiliza para colocar el título pricipal de nivel 1
  • <h2></h2> etiqueta que se utiliza para colocar títulos de nivel 2
  • <h3></h3> etiqueta que se utiliza para colocar títulos de nivel 3
  • <h4></h4> etiqueta que se utiliza para colocar títulos de nivel 4
  • <h5></h5> etiqueta que se utiliza para colocar títulos de nivel 5
  • <h6></h6> etiqueta que se utiliza para colocar títulos de nivel 6

PARRAFOS

Para los párrafos HTML proporciona las etiquetas. <p></p> Cada párrafo de la página se debe encerrar entre este par de etiquetas y entre cada párrafo se mostrará un espaciado.

  • <p></p>La etiqueta p (párrafo) es el apropiado para distribuir el texto en párrafos.
  • <br> Se utiliza para crear espacio entre parrafos

HIPERENLACES

es un elemento de un documento electrónico que hace referencia a otro recurso, como por ejemplo un punto específico de un documento o de otro documento.

  • <a></a> Representa un hiperenlace, a este le podemos agregar un nombre. Tenemos que utilizar el atributo href.(href="")

TABLAS

se utiliza el atributo colspan para unir casillas. Se utiliza el atributo rowspan para unir columnas.

  • <table></table> La etiqueta table define una tabla. Una tabla va a poder tener diferentes filas y dentro de cada fila vamos a poder definir diferentes celdas.
  • <tr></tr> Se utiliza para indecar las filas
  • <td></td> Se utiliza para indecar las celdas

LISTADOS

  • <ul></ul> etiqueta que se utiliza para crear las viñetas
  • <ol></ol> etiqueta que se utiliza para crear un listado ordenado
  • <li></li> etiqueta que se utiliza para crear un listado no ordenado

LISTA DE ETIQUETAS DIRIGIDAS A OBJETOS

ESTRUCURAS

Las etiquetas de estructura son de tipo objeto ya que pueden ser redimensionadas tanto en ancho y en alto estas etiquetas también son conocidas como cajas.

  • <html></html> El texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto.

  • <header></header> Se utiliza para crear un encabezado

  • <nav></nav>representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos.

  • <div></div>Sirve para crear secciones o agrupar contenidos.

  • <section><section>representa una sección genérica de un documento. Sirve para determinar qué contenido corresponde a qué parte de un esquema.

  • <footer></footer> representa un pie de página para el contenido de sección más cercano o el elemento raíz de sección

IMAGENES

  • <img></img> Se utiliza para colocar una imagen acompañado del atributo src que indica la dirección de la imagen.

Se coloca el argumento rel="stylesheet" seguido del argumento href="css/css.css (carpeta donde se encuentra el documento/css.css) para una hoja de estilo en cascada. Esta etiqueta solo se apertura.
